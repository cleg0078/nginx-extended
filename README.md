# Build nginx with extra modules 
## Created and used for BitsaboutMe project

* http3 support
* brotli compression
* cache_purge
* geoip2
* naxsi
* headers_more
* ModSecurity
* njs
* cookie_flag

## runs for ubuntu & debian 

All-in-one compilation in a single script

Just launch the script

```
./build.sh
```
